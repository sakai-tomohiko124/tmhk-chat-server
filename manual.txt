TMHKchat アプリ説明書

■ はじめに
このアプリは、多目的なコミュニケーションを支援するためのチャットアプリケーションです。
友達との会話から、仕事の連携、自分だけのメモまで、幅広くご活用いただけます。

■ 主な機能
1. トーク機能
   - 1対1のプライベートチャットが可能です。
   - 複数人で会話できるグループチャットを作成できます。

2. タイムライン
   - いま何をしているか、写真や動画と一緒に投稿できます。
   - 友達の投稿をチェックしましょう。

3. ミニゲーム
   - 友達と簡単なゲームで遊ぶことができます。息抜きにどうぞ。

4. AIチャット
   - 最新のAI（Gemini）と自由に会話できます。調べ物やアイデア出しに便利です。

■ 設定について
「その他」タブから、各種設定を変更できます。
- プロフィール編集: アイコンや自己紹介文を変更できます。
- 通知設定: 通知のオン・オフを切り替えられます。
- パスワード変更: 定期的なパスワードの変更を推奨します。

■ 困ったときは
管理者までお気軽にお問い合わせください。

## 🎯 プロジェクト概要

### 📊 基本情報

| 項目 | 詳細 |
|------|------|
| **アプリ名** | TMchat |
| **主要プラットフォーム** | スマートフォン（メイン）、PC（サブ） |
| **技術スタック** | Python(Flask) + HTML + CSS + JavaScript |
| **データベース** | SQLite（開発）→ AWS RDS（本番） |
| **クラウド** | AWS（無料枠のみ） |
| **AI** | Google AI Studio（無料枠） |
| **対象ユーザー** | 職場・家族・プライベート全対応 |

### 🎨 参考アプリ機能統合

- **LINE**: スタンプ、グループチャット、通話機能
- **Instagram**: ストーリー、写真・動画共有、フィルター
- **TikTok**: ショート動画、エフェクト、音楽同期
- **WeChat**: ミニプログラム、決済機能、公式アカウント
- **Slack**: ワークスペース、スレッド、ボット統合
